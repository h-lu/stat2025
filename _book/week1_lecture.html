第一周：启程：统计思维、R与数据世界

1. 欢迎来到统计学与R的世界！

本周我们将开启一段探索数据奥秘的旅程。统计学不仅仅是数学公式，更是一种理解世界、做出明智决策的思维方式。而 R 语言，则是我们探索数据、实践统计思维的强大工具。

本周目标
  • 理解统计学的基本概念和重要性。
  • 熟悉 R 和 RStudio 的基本操作环境。
  • 掌握 R 的基础语法，为后续学习打下基础。
  • 初步了解 AI 助手在学习和数据分析中的潜力。
  • 明确课程目标、学习方式和最终项目要求。

2. 统计思维：超越直觉

  • 什么是统计学？
    • 收集、处理、分析、解释数据，并从中得出结论的科学。
    • 在不确定性中寻找规律，量化证据。
  • 为何学习统计学？
    • 培养数据驱动的决策能力。
    • 批判性地评估信息和研究。
    • 解决现实世界中的复杂问题（商业、科研、医疗等）。
  • 统计思维的核心：
    • 变异性 (Variation): 认识到数据总是存在差异。
    • 随机性 (Randomness): 理解概率在抽样和推断中的作用。
    • 模型 (Models): 使用简化的数学表示来理解复杂现象。
    • 推断 (Inference): 从样本信息推广到总体特征。

3. R 与 RStudio：你的数据科学工作台

  • R 语言简介:
    • 开源、免费、强大的统计计算和图形绘制语言。
    • 拥有庞大的社区和丰富的扩展包 (Packages)。
  • RStudio IDE:
    • 集成开发环境 (Integrated Development Environment)，让 R 的使用更便捷。
    • 主要窗口：脚本编辑器、控制台、环境/历史记录、文件/图形/包/帮助。
  • 环境搭建:
    • 安装 R: https://cran.r-project.org/
    • 安装 RStudio Desktop: https://posit.co/download/rstudio-desktop/
  • tidyverse 包介绍与安装:
    • tidyverse 是一个包含了一系列用于数据科学的 R 包的集合（如 ggplot2, dplyr, tidyr, readr 等），它们共享一致的设计哲学、语法和数据结构。
    • 安装命令：
    # install.packages("tidyverse") # 如果尚未安装，取消注释并运行
library(tidyverse) # 加载包

4. R 基础语法入门

  • R 作为计算器:

    1 + 1
5 * 3
10 / 2
2 ^ 3 # 幂运算

  • 变量赋值: 使用 <- 或 = (推荐 <-)

    x <- 5
y <- x * 2
x
y

  • 数据类型初识:

    • numeric: 数值型 (包括整数和浮点数)
    • character: 字符型 (文本，用引号括起来)
    • logical: 逻辑型 (TRUE 或 FALSE)
    a <- 10.5
b <- "Hello, R!"
c <- TRUE
class(a) # 查看变量类型
class(b)
class(c)

  • 向量 (Vector): R中最基本的数据结构，存储相同类型元素的序列。使用 c() 函数创建。

    numeric_vector <- c(1, 3, 5, 7)
char_vector <- c("apple", "banana", "cherry")
logical_vector <- c(TRUE, FALSE, TRUE, TRUE)

numeric_vector
char_vector[1] # 访问第一个元素 (R的索引从1开始)
length(numeric_vector) # 查看向量长度
    向量类型一致性

    一个向量中的所有元素必须是相同的数据类型。如果混合不同类型，R 会进行强制类型转换（通常转换为字符型）。

    mixed_vector <- c(1, "two", TRUE)
mixed_vector # 查看结果
class(mixed_vector)

5. AI 助手初识：你的智能学习伙伴

现代 AI 工具（如 ChatGPT、Copilot 等）可以在学习 R 和统计学过程中提供巨大帮助。

AI 辅助演示
  • 查询 R 语法: “如何在 R 中创建数值向量？” 或 “解释 R 中 class() 函数的作用。”
  • 解释统计概念: “请解释统计学中变异的概念” 或 “平均数和中位数有什么区别？”
  • 代码调试: (后续会涉及) 粘贴错误信息，询问可能的错误原因。
  • 生成示例代码: “请给我一个在 R 中使用 dplyr 包 filter() 函数的示例。”

重要提示: AI 是强大的助手，但并非绝对权威。务必批判性地看待 AI 的回答，并通过官方文档、书籍或老师进行验证。 它的主要价值在于快速获取信息、启发思路和辅助编程，而非替代独立思考和深入理解。

6. 课程与项目介绍

  • 课程目标: 掌握数据处理、可视化和常用统计推断方法，能够使用 R 语言独立完成基本的数据分析任务。
  • 教学方式: 理论讲解 + R 代码演示 + 案例分析 + 动手练习 + Capstone 项目。
  • 评估方式: 平时作业、期中/期末考试、Capstone 项目。
  • Capstone 项目: (后续详细介绍) 一个综合性的数据分析项目，要求应用课程所学知识解决一个实际或模拟的问题。
