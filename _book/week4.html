第七次课：参数估计、假设检验原理

推断统计初步：参数估计

在之前的课程中，我们主要学习了描述统计，即如何用图表和统计量描述样本数据的特征。从本周开始，我们将进入推断统计的学习。推断统计的核心思想是用样本数据推断总体特征。

参数估计是推断统计的重要内容之一，它的目的是用样本统计量估计总体参数。

描述统计 vs. 推断统计

  • 描述统计 (Descriptive Statistics):
    • 目的: 描述和总结样本数据的特征。
    • 方法: 计算描述性统计量 (均值、中位数、标准差等)，绘制图表 (直方图、箱线图、散点图等)。
    • 结论: 结论仅限于样本数据本身，不能推广到总体。
    • 例子: 计算样本电影评分的平均分，绘制样本电影评分的分布图。
  • 推断统计 (Inferential Statistics):
    • 目的: 利用样本数据推断总体特征。
    • 方法: 参数估计 (点估计、区间估计)，假设检验。
    • 结论: 结论可以推广到总体，但存在不确定性。
    • 例子: 用样本电影评分的平均分估计所有电影的平均评分，检验总体电影评分的平均分是否高于某个值。

参数估计的目的

  • 总体 (Population): 我们感兴趣的所有个体的集合。例如，所有电影，所有大学生，所有股票。

  • 参数 (Parameter): 描述总体特征的数值，通常是未知的，需要用样本数据估计。例如，总体均值 μ，总体标准差 σ，总体比例 p。

  • 样本 (Sample): 从总体中随机抽取的一部分个体。

  • 统计量 (Statistic): 描述样本特征的数值，可以根据样本数据计算得到，用于估计总体参数。例如，样本均值 \(\bar{x}\)，样本标准差 s，样本比例 \(\hat{p}\)。

  • 参数估计 (Parameter Estimation): 用样本统计量估计总体参数 的过程。

    • 点估计 (Point Estimation): 用一个样本统计量的值作为总体参数的估计值。
    • 区间估计 (Interval Estimation): 给出一个总体参数的可能取值范围，通常以置信区间的形式呈现。

点估计 (Point Estimation)

  • 点估计: 用一个样本统计量的值直接作为总体参数的估计值。

    • 用样本均值 \(\bar{x}\) 估计总体均值 μ。
    • 用样本比例 \(\hat{p}\) 估计总体比例 p。
    • 用样本标准差 s 估计总体标准差 σ。

  • 常用点估计量:

    • 样本均值 \(\bar{x}\): 总体均值 μ 的点估计。
    • 样本中位数 Median: 总体中位数的点估计。
    • 样本标准差 s: 总体标准差 σ 的点估计。
    • 样本比例 \(\hat{p}\): 总体比例 p 的点估计。

  • 点估计量的评价标准 (概念性了解):

    • 无偏性 (Unbiasedness): 估计量的期望值等于总体参数的真值。
    • 有效性 (Efficiency): 对于同一个总体参数，多个无偏估计量中，方差最小的估计量更有效。
    • 一致性 (Consistency): 随着样本量增大，估计量的值越来越接近总体参数的真值。

  • R 语言实现点估计: 使用 R 函数直接计算样本统计量，作为总体参数的点估计值。

    # 示例数据 (电影评分)
movie_ratings <- c(4.5, 4.2, 3.8, 4.7, 4.0, 4.3, 3.9, 4.6, 4.1, 4.4)

# 计算样本均值 (点估计总体均值)
mean_rating_point_estimate <- mean(movie_ratings)
mean_rating_point_estimate
    [1] 4.25
    # 计算样本标准差 (点估计总体标准差)
sd_rating_point_estimate <- sd(movie_ratings)
sd_rating_point_estimate
    [1] 0.302765
    # 计算样本中位数 (点估计总体中位数)
median_rating_point_estimate <- median(movie_ratings)
median_rating_point_estimate
    [1] 4.25

区间估计 (Interval Estimation)

  • 为什么需要区间估计？: 点估计存在误差。点估计只给出一个具体的数值，无法反映估计的精确程度。区间估计提供一个参数的可能取值范围，并给出置信水平，能够反映估计的不确定性。

  • 置信区间 (Confidence Interval, CI): 由样本数据计算出的，包含总体参数真值的可能性为 \((1-\alpha)\) 的区间。

    • 置信水平 (Confidence Level, \(1-\alpha\)): 表示置信区间包含总体参数真值的概率。常用的置信水平为 95% ( \(\alpha = 0.05\) ) 和 99% ( \(\alpha = 0.01\) )。
    • 置信区间的宽度: 置信区间的上限减去下限。宽度越小，估计越精确。

  • 影响置信区间宽度的因素:

    • 样本量 (n): 样本量越大，置信区间越窄。
    • 置信水平 (\(1-\alpha\)): 置信水平越高，置信区间越宽。
    • 总体标准差 (σ) 或样本标准差 (s): 标准差越大，置信区间越宽。

  • 正态分布的置信区间 (总体均值 μ 的置信区间):

    • 已知总体标准差 σ 或大样本情况 (n ≥ 30)，可以使用 Z 分布 构建置信区间。

    • 总体均值 μ 的 \((1-\alpha)\) 置信区间为: \[ \bar{x} \pm Z_{\alpha/2} \frac{\sigma}{\sqrt{n}} \] 其中，\(\bar{x}\) 为样本均值，\(\sigma\) 为总体标准差 (或用样本标准差 s 近似)，n 为样本量，\(Z_{\alpha/2}\) 为标准正态分布的 \(1-\alpha/2\) 分位数。

    • 未知总体标准差 σ 且小样本情况 (n < 30)，如果总体服从正态分布，可以使用 t 分布 构建置信区间。

    • 总体均值 μ 的 \((1-\alpha)\) 置信区间为: \[ \bar{x} \pm t_{\alpha/2, n-1} \frac{s}{\sqrt{n}} \] 其中，\(\bar{x}\) 为样本均值，s 为样本标准差，n 为样本量，\(t_{\alpha/2, n-1}\) 为自由度为 n-1 的 t 分布的 \(t_{\alpha/2, n-1}\) 分位数。

    • R 语言实现区间估计: 手动计算置信区间的公式。

      # 示例数据 (电影评分)
movie_ratings <- c(4.5, 4.2, 3.8, 4.7, 4.0, 4.3, 3.9, 4.6, 4.1, 4.4)
sample_mean <- mean(movie_ratings)
sample_sd <- sd(movie_ratings)
sample_size <- length(movie_ratings)
confidence_level <- 0.95
alpha <- 1 - confidence_level

# 查 t 分布临界值 (双尾)
t_critical <- qt(1 - alpha/2, df = sample_size - 1)

# 计算标准误差 (Standard Error, SE)
standard_error <- sample_sd / sqrt(sample_size)

# 计算边际误差 (Margin of Error, ME)
margin_of_error <- t_critical * standard_error

# 计算置信区间
confidence_interval <- c(sample_mean - margin_of_error, sample_mean + margin_of_error)
confidence_interval
      [1] 4.033415 4.466585
      # 解释置信区间
cat(paste("总体均值", confidence_level*100, "% 的置信区间为:",
          round(confidence_interval[1], 2), "到", round(confidence_interval[2], 2)))
      总体均值 95 % 的置信区间为: 4.03 到 4.47

  • 参数估计练习:

    • 练习任务： 提供一个示例数据集 (例如，电影评分数据)。
    • 计算样本均值和样本标准差，作为总体均值和总体标准差的点估计值。
    • 手动计算总体均值的 95% 和 99% 置信区间 (假设总体服从正态分布，使用样本标准差近似总体标准差)。
    • 解释置信区间的含义。
    • 鼓励学生使用AI工具 (如Cursor) 辅助公式计算和R代码编写。 例如，让AI生成计算置信区间的R代码，或者解释置信区间的公式。
推断统计初步：假设检验

假设检验 (Hypothesis Testing) 是推断统计的另一个重要内容。它的目的是检验关于总体参数的假设是否成立。

假设检验的目的

  • 研究问题: 研究者通常会针对某个总体参数提出一个假设，例如：
    • 某电影类型的平均评分是否高于 4.0 分？
    • 不同类型电影的平均评分是否相等？
    • 广告投放后，产品销量是否显著增加？
  • 假设检验的目的: 通过样本数据提供的证据，判断提出的假设是否合理。

假设检验的基本思想

  • 反证法: 先提出一个与研究目的相反的假设 (原假设)，然后试图用样本数据反驳原假设。
  • 小概率事件原理: 如果原假设成立，某个事件发生的概率很小 (小概率事件)，但在一次试验中，该事件竟然发生了，那么我们就有理由怀疑原假设的真实性，从而拒绝原假设。
  • 逻辑:
    • 假设 \(H_0\) 成立 (原假设) \(\Rightarrow\) 小概率事件 A 发生 \(\Rightarrow\) 拒绝 \(H_0\)。

假设检验的步骤

  1. 提出原假设 (Null Hypothesis, \(H_0\)) 和备择假设 (Alternative Hypothesis, \(H_1\))。
  2. 选择检验统计量 (Test Statistic)。
  3. 确定显著性水平 (Significance Level, \(\alpha\), 例如 0.05, 0.01)。
  4. 计算检验统计量的 p 值 (p-value)。
  5. 做出决策： 如果 p-value \(\leq \alpha\)，拒绝原假设 \(H_0\)，接受备择假设 \(H_1\)；如果 p-value \(> \alpha\)，不拒绝原假设 \(H_0\)。

原假设 (\(H_0\)) 和备择假设 (\(H_1\))

  • 原假设 (\(H_0\)): 研究者想要检验的假设，通常是”没有效应”、“没有差异”、“没有关系”的假设。
    • 例如： 总体均值等于某个特定值 (\(\mu = \mu_0\))，两个总体均值相等 (\(\mu_1 = \mu_2\))，变量之间没有关系 (相关系数为 0)。
    • 总是关于总体参数的假设，而不是关于样本统计量的假设。
    • 在假设检验的框架下，总是先假定原假设 \(H_0\) 成立。
  • 备择假设 (\(H_1\)): 研究者希望证明的假设，通常是”存在效应”、“存在差异”、“存在关系”的假设。
    • 例如： 总体均值不等于某个特定值 (\(\mu \neq \mu_0\))，总体均值大于某个特定值 (\(\mu > \mu_0\))，两个总体均值不相等 (\(\mu_1 \neq \mu_2\))，变量之间存在关系 (相关系数不为 0)。
    • 与原假设 \(H_0\) 互斥，即 \(H_0\) 和 \(H_1\) 不可能同时成立。
    • 备择假设的形式:
      • 双尾检验 (two-tailed test): \(H_1: \mu \neq \mu_0\) (不等于)。 检验总体均值是否不同于某个特定值，不关心是大于还是小于。
      • 右尾检验 (right-tailed test): \(H_1: \mu > \mu_0\) (大于)。 检验总体均值是否显著大于某个特定值。
      • 左尾检验 (left-tailed test): \(H_1: \mu < \mu_0\) (小于)。 检验总体均值是否显著小于某个特定值。
    • 研究者需要根据研究问题和目的，选择合适的备择假设形式。
  • 原假设和备择假设的设定原则:
    • 互相排斥: \(H_0\) 和 \(H_1\) 必须是互相排斥的，不能同时成立。
    • 完备性: \(H_0\) 和 \(H_1\) 应该覆盖所有可能的情况。
    • 优先保护原假设: 在没有充分证据的情况下，我们倾向于不拒绝原假设 \(H_0\)。 只有当样本数据提供足够强的证据时，我们才拒绝 \(H_0\)，接受 \(H_1\)。

显著性水平 (\(\alpha\))

  • 显著性水平 (\(\alpha\)): 预先设定的一个概率值，作为判断小概率事件的标准。
    • 常用取值: \(\alpha = 0.05\) (5%), \(\alpha = 0.01\) (1%), \(\alpha = 0.10\) (10%)。
    • \(\alpha = 0.05\) 的含义: 如果原假设 \(H_0\) 成立，拒绝 \(H_0\) 的概率为 5%。 也就是说，在 100 次假设检验中，平均有 5 次会犯第一类错误 (拒真错误)。
    • \(\alpha\) 越小，拒绝 \(H_0\) 的标准越严格，犯第一类错误的概率越小。
    • \(\alpha\) 的选择需要根据研究的具体情况和后果来决定。 一般来说，探索性研究可以适当放宽 \(\alpha\)，验证性研究和高风险决策需要严格控制 \(\alpha\)。

p 值 (p-value)

  • p 值 (p-value): 在原假设 \(H_0\) 成立的条件下，观察到样本结果或更极端结果的概率。
    • “更极端”: 指更有利于备择假设 \(H_1\) 的样本结果。 例如，对于右尾检验 (\(H_1: \mu > \mu_0\))，样本均值 \(\bar{x}\) 越大，就越极端。
    • p 值的意义: 衡量样本数据与原假设 \(H_0\) 的不一致程度。 p 值越小，说明样本数据与原假设 \(H_0\) 越不一致，拒绝 \(H_0\) 的理由越充分。
    • p 值是一个概率值，取值范围为 [0, 1]。

决策规则

  • 比较 p 值和显著性水平 \(\alpha\):
    • 如果 p-value \(\leq \alpha\): 拒绝原假设 \(H_0\)，接受备择假设 \(H_1\)。 结论为“在 \(\alpha\) 显著性水平下，拒绝原假设 $H_0”，或“在 \(\alpha\) 显著性水平下，存在统计学显著的效应/差异/关系”。
    • 如果 p-value \(> \alpha\): 不拒绝原假设 \(H_0\)。 结论为“在 \(\alpha\) 显著性水平下，不拒绝原假设 $H_0”，或“在 \(\alpha\) 显著性水平下，没有足够证据拒绝原假设 $H_0”，或“不能得出统计学显著的效应/差异/关系”。
    • 注意: “不拒绝原假设 $H_0” 不等于 “接受原假设 $H_0”。 我们只能说没有足够证据拒绝 $H_0”，但不能说 \(H_0\) 一定成立。 假设检验只能证伪，不能证真。

两类错误

  • 第一类错误 (Type I error, 拒真错误): 原假设 \(H_0\) 实际上是成立的，但我们却拒绝了 \(H_0"**。 * **概率**: 犯第一类错误的概率为 **\)$ (显著性水平)。 这是我们可以控制的错误概率。

    • 例子: 本来某电影类型的平均评分没有高于 4.0 分 (\(H_0\) 成立)，但假设检验结果却错误地认为该类型电影的平均评分高于 4.0 分 (拒绝 \(H_0\))。

  • 第二类错误 (Type II error, 受伪错误): 原假设 \(H_0\) 实际上是不成立的，但我们却没有拒绝 \(H_0"**。 * **概率**: 犯第二类错误的概率为 **\)\(**，**通常是未知的，难以直接控制**。 * **例子**: 本来某电影类型的平均评分**确实**高于 4.0 分 (\)H_0$ 不成立)，但假设检验结果却错误地认为该类型电影的平均评分没有**高于 4.0 分 (不拒绝 \(H_0\))。

  • 两类错误的权衡:

    • \(\alpha\) 和 \(\beta\) 之间存在权衡关系。 减小 \(\alpha\) (更严格地拒绝 \(H_0\))，会增大 \(\beta\) (更容易犯第二类错误)； 增大 \(\alpha\) (更宽松地拒绝 \(H_0\))，会减小 \(\beta\) (更不容易犯第二类错误)。
    • 在实际研究中，需要根据研究目的和后果，权衡两类错误的重要性，选择合适的 \(\alpha\) 水平。 一般来说，更重视控制第一类错误，通常取 \(\alpha = 0.05\)。

效应量 (Effect Size)

  • 效应量 (Effect Size): 反映效应/差异/关系 的真实大小的指标，不受到样本量大小的影响。
    • 与 p 值不同: p 值只反映统计学显著性，不反映效应的实际大小。 即使 p 值很小 (统计学显著)，效应量也可能很小 (实际意义不大)。 反之，即使 p 值较大 (统计学不显著)，效应量也可能很大 (但由于样本量不足，未能达到统计学显著)。
    • 常用的效应量指标:
      • Cohen’s d: 用于衡量两组均值差异的效应量，常用于 t 检验。 Cohen’s d = (Mean1 - Mean2) / Pooled SD。
      • 相关系数 r: 用于衡量两个变量之间线性关系强度的效应量，取值范围为 [-1, 1]。
      • R 平方 (\(R^2\)): 用于衡量回归模型解释变异的比例的效应量，取值范围为 [0, 1]。
    • 效应量大小的解释 (Cohen’s d 为例):
      • 小效应 (small effect): Cohen’s d ≈ 0.2
      • 中等效应 (medium effect): Cohen’s d ≈ 0.5
      • 大效应 (large effect): Cohen’s d ≈ 0.8
    • 效应量的大小解释标准是相对的，需要结合具体的研究领域和背景进行判断。

检验功效 (Power)

  • 检验功效 (Power, \(1-\beta\)): 当备择假设 \(H_1\) 实际上成立时，我们正确地拒绝原假设 \(H_0\) 的概率。

    • 与第二类错误 (\(\beta\)) 的关系: Power = \(1 - \beta\)。 Power 越高，犯第二类错误的概率 \(\beta\) 越小。
    • Power 的意义: 衡量假设检验的灵敏度。 Power 越高，越容易发现真实存在的效应/差异/关系。 Power 太低，即使真实存在效应，也可能由于样本量不足等原因，无法通过假设检验发现 (导致不拒绝 \(H_0\))。
    • 一般认为，Power 至少要达到 0.8 (80%) 以上，才能认为检验是有效的。

  • 影响检验功效的因素:

    • 效应量 (Effect Size): 效应量越大，Power 越高。效应越大，越容易被检验出来。
    • 样本量 (Sample Size, n): 样本量越大，Power 越高。样本量越大，估计越精确，越容易发现真实效应。
    • 显著性水平 (\(\alpha\)): \(\alpha\) 越大，Power 越高。\(\alpha\) 越大，拒绝 \(H_0\) 的标准越宽松，Power 越高，但犯第一类错误的概率也增大。
    • 总体标准差 (\(\sigma\)): 总体标准差越小，Power 越高。标准差越小，数据变异性越小，越容易发现真实效应。
    • 单尾检验 vs. 双尾检验: 单尾检验的 Power 通常高于双尾检验 (在效应方向与检验方向一致时)。

  • 检验功效分析 (Power Analysis): 在研究设计阶段，为了达到一定的 Power 水平，需要计算所需的最小样本量。

    • 先设定期望的 Power 值 (如 0.8)，以及预期的效应量大小、显著性水平 \(\alpha\)、总体标准差 (或估计值)。
    • 然后，使用统计软件或公式计算所需的最小样本量。
    • 检验功效分析有助于研究者合理规划研究设计，避免样本量不足导致无法发现真实效应。

  • R 语言中进行功效分析: 可以使用 pwr 包进行功效分析。

上半场 (单样本t检验、双样本t检验，约45分钟):

  • 常用假设检验方法：t检验 (30分钟): {#week-4-t-test}
    • t检验的应用场景： 检验总体均值是否等于某个值，或检验两个总体的均值是否相等。
    • 单样本t检验 (One-Sample t-test)：

      • 适用条件： 检验一个正态分布总体的均值是否等于某个已知值。

      • 原假设 \(H_0：\mu = \mu_0\)，备择假设 \(H_1：\mu \neq \mu_0\) (双尾检验), \(\mu > \mu_0\) (右尾检验), \(\mu < \mu_0\) (左尾检验)。

      • 检验统计量：t统计量公式。 \[ t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}} \] 其中，\(\bar{x}\) 为样本均值，\(\mu_0\) 为原假设下的总体均值，\(s\) 为样本标准差，\(n\) 为样本量。

      • 自由度 (degrees of freedom, df) = n - 1。

      • p值的计算：使用t分布的累积分布函数 (CDF) 或生存函数 (SF)。

      • R语言实现： t.test() 函数是 R 中进行 t 检验的核心函数。

        • 基本用法： t.test(data, mu = ?, alternative = ?, conf.level = ?, ...)

        • 主要参数：

          • data: 必需参数。 包含样本数据的数值向量。
          • mu: 单样本 t 检验时使用。 原假设中总体均值 \(\mu_0\) 的值。 默认为 mu = 0。
          • alternative: 备择假设类型。 字符串，可选值包括：
            • "two.sided" (默认值): 双尾检验，\(H_1: \mu \neq \mu_0\)。
            • "less": 左尾检验，\(H_1: \mu < \mu_0\)。
            • "greater": 右尾检验，\(H_1: \mu > \mu_0\)。
          • conf.level: 置信水平。 数值，介于 0 和 1 之间，表示置信区间的置信水平。 默认为 conf.level = 0.95 (95% 置信区间)。
          • paired: 配对样本 t 检验时使用。 逻辑值，TRUE 表示进行配对样本 t 检验，FALSE (默认值) 表示进行独立样本 t 检验。
          • var.equal: 独立样本 t 检验时使用。 逻辑值，TRUE 表示假设两个总体的方差相等 (等方差 t 检验)，FALSE (默认值) 表示不假设方差相等 (Welch’s t 检验)。

        • 示例 (单样本 t 检验):

          # 示例数据 (电影评分)
movie_ratings <- c(4.5, 4.2, 3.8, 4.7, 4.0, 4.3, 3.9, 4.6, 4.1, 4.4)

# 进行单样本 t 检验，检验均值是否为 4.0
t.test(movie_ratings, mu = 4.0)

    • 双样本t检验 (Two-Sample t-test)：
      • 独立样本t检验 (Independent Samples t-test)：

        • 适用条件： 检验两个独立的正态分布总体的均值是否相等。例如，比较不同类型电影的平均评分，不同超市的商品平均价格，不同学校学生的平均成绩等。

        • 原假设 \(H_0：\mu_1 = \mu_2\)，备择假设 \(H_1：\mu_1 \neq \mu_2\) (双尾检验), \(\mu_1 > \mu_2\) (右尾检验), \(\mu_1 < \mu_2\) (左尾检验)。

        • 检验统计量：独立样本t统计量公式 (等方差和不等方差两种情况)。

          • 等方差 (Pooled Variance) t 检验: 假设两个总体的方差相等 (\(\sigma_1^2 = \sigma_2^2\))。 \[ t = \frac{(\bar{x}_1 - \bar{x}_2) - (\mu_1 - \mu_2)}{s_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \] 其中：
            • \(\bar{x}_1, \bar{x}_2\) 分别为两个样本的均值。
            • \(\mu_1 - \mu_2\) 为原假设下两个总体均值之差，通常为 0。
            • \(n_1, n_2\) 分别为两个样本的样本量。
            • \(s_p\) 为合并样本标准差 (pooled standard deviation)，计算公式为： \[ s_p = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1 + n_2 - 2}} \] 其中 \(s_1^2, s_2^2\) 分别为两个样本的方差。
          • 不等方差 (Welch’s) t 检验: 不假设两个总体的方差相等 (\(\sigma_1^2 \neq \sigma_2^2\))。 \[ t = \frac{(\bar{x}_1 - \bar{x}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}} \] 其中：
            • \(\bar{x}_1, \bar{x}_2\) 分别为两个样本的均值。
            • \(\mu_1 - \mu_2\) 为原假设下两个总体均值之差，通常为 0。
            • \(n_1, n_2\) 分别为两个样本的样本量。
            • \(s_1^2, s_2^2\) 分别为两个样本的方差。

        • 自由度 (df) 的计算 (等方差和不等方差两种情况)。

        • 方差相等性检验: 在进行独立样本 t 检验时，需要考虑两个总体的方差是否相等。

      等方差假设 (Equal Variance)

      如果假设两个总体的方差相等 (\(\sigma_1^2 = \sigma_2^2\))，则使用等方差 t 检验 (Pooled Variance t-test)。

      适用情况

      当事先有理由相信两个总体的方差接近，或者通过方差相等性检验 (如 F 检验或 Levene 检验) 结果不拒绝方差相等的原假设时，可以使用等方差 t 检验。

      不等方差假设 (Unequal Variance)

      如果不能假设两个总体的方差相等 (\(\sigma_1^2 \neq \sigma_2^2\))，或者方差相等性检验拒绝了方差相等的原假设，则使用 Welch’s t 检验 (Welch’s t-test)。

      适用情况

      当不确定两个总体的方差是否相等，或者怀疑方差存在显著差异时，建议使用 Welch’s t 检验。 Welch’s t 检验对总体方差是否相等不敏感，适用性更广，通常作为默认选择。

      R 语言 t.test() 函数
      通过 var.equal 参数指定是否假设方差相等。
      • var.equal = TRUE: 进行等方差 t 检验。
      • var.equal = FALSE (默认值): 进行 Welch’s t 检验。
      • 配对样本t检验 (Paired Samples t-test)：
        • 适用条件： 检验两个配对样本的均值差是否为零。例如，检验同一组用户对电影观看预告片前后的评分是否有显著差异。
        • 原假设 \(H_0：\mu_d = 0\)，备择假设 \(H_1：\mu_d \neq 0\) (双尾检验), \(\mu_d > 0\) (右尾检验), \(\mu_d < 0\) (左尾检验)，其中 \(\mu_d\) 是配对样本差值的总体均值。
        • 检验统计量：配对样本t统计量公式： \[ t = \frac{\bar{d} - \mu_{d0}}{s_d / \sqrt{n}} \] 其中：
          • \(\bar{d}\) 为配对样本差值的均值。
          • \(\mu_{d0}\) 为原假设下配对样本差值的总体均值，通常为 0。
          • \(s_d\) 为配对样本差值的标准差。
          • \(n\) 为配对样本的数量。
        • 自由度 (df) = n - 1，其中 n 是配对样本的数量。
        • R语言实现： t.test() 函数，设置 paired = TRUE。

参数估计的关键概念
  • 点估计: 用一个样本统计量的值直接作为总体参数的估计值
  • 区间估计: 给出一个总体参数的可能取值范围，通常以置信区间的形式呈现
  • 置信区间: 由样本数据计算出的，包含总体参数真值的可能性为 \((1-\alpha)\) 的区间
假设检验的五个步骤
  1. 提出原假设 (\(H_0\)) 和备择假设 (\(H_1\))
  2. 选择检验统计量
  3. 确定显著性水平 (\(\alpha\))
  4. 计算检验统计量的 p 值
  5. 做出决策：
    • 如果 p-value ≤ α，拒绝 \(H_0\)，接受 \(H_1\)
    • 如果 p-value > α，不拒绝 \(H_0\)
两类错误与检验功效
  • 第一类错误 (Type I): 错误地拒绝真实的 \(H_0\)，概率为 α
  • 第二类错误 (Type II): 错误地接受错误的 \(H_0\)，概率为 β
  • 检验功效 (Power): 正确拒绝 \(H_0\) 的概率，等于 1-β
t检验的三种类型
  1. 单样本t检验
    • 检验一个正态分布总体的均值是否等于某个已知值
    • \(H_0：\mu = \mu_0\) vs \(H_1：\mu \neq \mu_0\) (或 > 或 <)
  2. 独立样本t检验
    • 检验两个独立的正态分布总体的均值是否相等
    • \(H_0：\mu_1 = \mu_2\) vs \(H_1：\mu_1 \neq \mu_2\) (或 > 或 <)
  3. 配对样本t检验
    • 检验配对样本的差异是否显著
    • \(H_0：\mu_d = 0\) vs \(H_1：\mu_d \neq 0\) (或 > 或 <)
置信区间的计算公式

  1. 大样本情况 (n ≥ 30) 或已知总体标准差: \[ \bar{x} \pm Z_{\alpha/2} \frac{\sigma}{\sqrt{n}} \]

  2. 小样本情况 (n < 30) 且未知总体标准差: \[ \bar{x} \pm t_{\alpha/2, n-1} \frac{s}{\sqrt{n}} \]

假设检验的基本思想
  • 反证法: 先提出一个与研究目的相反的假设(原假设)
  • 小概率事件原理: 如果原假设成立，某事件发生概率很小，但该事件发生了
  • 逻辑: \(H_0\) 成立 → 小概率事件发生 → 拒绝 \(H_0\)
原假设和备择假设的设定原则
  1. 互相排斥: \(H_0\) 和 \(H_1\) 不能同时成立
  2. 完备性: 覆盖所有可能的情况
  3. 优先保护原假设: 没有充分证据时不拒绝 \(H_0\)
影响检验功效的因素
  • 效应量: 效应量越大，Power越高
  • 样本量: 样本量越大，Power越高
  • 显著性水平: α越大，Power越高
  • 总体标准差: 标准差越小，Power越高
  • 检验类型: 单尾检验的Power通常高于双尾检验
