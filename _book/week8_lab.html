第八周实验：阶段回顾与多元回归实践

1. 目标

本实验旨在巩固第一阶段的核心知识，并通过实践加深对多元线性回归 (MLR) 的理解，特别是偏回归系数的解释和模型基本假设的回顾。

  • 快速回顾 tidyverse 数据处理、ggplot2 可视化和第一阶段的关键统计方法。
  • 使用 lm() 拟合包含多个自变量的 MLR 模型。
  • 重点练习解读 MLR 模型 summary() 输出，特别是偏回归系数的含义（控制其他变量）。
  • 解释分类自变量（因子）在 MLR 中的系数含义。
  • 区分 R-squared 和 Adjusted R-squared。
  • 理解模型整体 F 检验的意义。
  • 回顾 L.I.N.E. 假设在线性回归中的重要性。

2. 阶段一快速回顾练习

在深入 MLR 之前，快速完成以下练习以巩固基础。

数据: 使用 mpg 数据集。

library(tidyverse)
library(car)  # 用于后续可能需要的方差膨胀因子(VIF)计算

练习题:

  1. dplyr: 计算每个 class (车辆类别) 的平均 cty (城市里程)，并按平均里程降序排列。
  2. ggplot2: 绘制 displ (排量) 与 cty 的散点图，并将点的颜色映射到 drv (驱动方式)，添加合适的标题和轴标签。
  3. 假设检验选择: 你想比较 drv 为 ‘f’ (前驱) 和 ‘r’ (后驱) 的车辆的平均 hwy (高速里程) 是否有显著差异，你会选择哪种统计检验？(写出名称即可)
  4. 相关性: 计算 cty 和 hwy 之间的 Pearson 相关系数。
  5. SLR 回顾: 拟合一个简单线性回归模型，用 cty 预测 hwy。写出回归方程，并解释斜率的含义。

3. 多元线性回归 (MLR) 实践

场景: 我们想建立一个更全面的模型来预测 mpg 数据集中的 hwy (高速公路里程)，同时考虑 cty (城市里程)、year (生产年份) 和 class (车辆类别) 的影响。

任务:

  1. 使用 lm() 拟合一个 MLR 模型，公式为 hwy ~ cty + as.factor(year) + class。将模型存储在 mlr_hwy 中。(注意：year 是二分类变量（1999 和 2008），使用 as.factor() 将其转换为因子变量)
  2. 使用 summary() 查看模型 mlr_hwy 的详细结果。
  3. 解读模型系数 (Coefficients):
    • 解释截距 (Intercept) 的含义（是否有实际意义？）。
    • 解释 cty 的偏回归系数的含义（强调“控制其他变量”）。
    • 解释 year 的偏回归系数的含义。(注意：year 的系数表示 2008 年相对于 1999 年（参照组）的 hwy 差异)
    • 解释 class 各水平的系数含义（它们是相对于哪个参照组的？）。
    • 判断每个系数的统计显著性（基于 P 值和 \(\alpha = 0.05\)）。
  4. 解读模型整体性能:
    • 解释 Adjusted R-squared 的值代表什么？
    • 解释 F-statistic 和对应的 p-value 的含义（检验模型的整体显著性）。

4. 统计显著性 vs 实际重要性讨论

思考:在上面的 mlr_hwy 模型中，year 的系数是统计显著的 (p ≈ 0.0036)，估计值为 0.46。

  1. 这个结果的统计意义是什么？
  2. 这个结果的实际重要性如何？增加约 0.46 MPG 对于购车者或汽车工程师来说，是一个值得关注的差异吗？（这取决于具体情境和比较基准）。
  3. 如果样本量增大很多，一个估计值为 0.1 MPG 的系数也可能变得统计显著 (p < 0.05)。在这种情况下，它的实际重要性又如何？

5. 回顾 L.I.N.E. 假设

回顾线性回归的四个关键假设：

  1. Linearity (线性性): Y 与 X 之间的关系是线性的。
  2. Independence (独立性): 误差项（残差）相互独立。
  3. Normality (正态性): 误差项（残差）服从正态分布。
  4. Equal Variance (等方差性 / Homoscedasticity): 误差项（残差）的方差在所有 X 水平上是恒定的。

思考: 对于我们拟合的 mlr_hwy 模型： * 你认为哪个假设最有可能被违反？为什么？ * 如果这些假设被违反，会对我们的模型结果（系数估计、P 值、置信区间）产生什么影响？（回顾讲义内容）。

（我们将在下周的实验中学习如何具体诊断这些假设。）

6. 实验总结

在本实验中，我们通过简短练习回顾了第一阶段的关键知识点。接着，我们重点实践了多元线性回归模型的拟合与解读，特别是深入理解了偏回归系数的含义——在控制其他变量影响下的独立效应。我们还练习了解读模型摘要中的 R² 和整体 F 检验，并讨论了统计显著性与实际重要性的区别。最后，我们回顾了 L.I.N.E. 假设，为下周的模型诊断做好铺垫。
